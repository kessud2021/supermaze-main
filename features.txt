SUPERMAZE v1.3.1 - FEATURES
============================

GAMEPLAY
--------
• Singleplayer Mode - Play custom random or imported mazes
• Daily Challenge (Seed System) - Everyone plays the same maze based on today's date
• Random Maze Generation - Procedurally generate new mazes
• Custom Maze Import - Load mazes from JSON or XML files
• Maze Solving - Navigate through mazes with arrow key controls
• Replay System - Save and replay completed game sessions
• Game Timing - Track completion time with millisecond precision (MM:SS.mmm format)

USER INTERFACE
--------------
• Username Authentication - Register and login with persistent session tokens
• Theme System - 6 preset themes (Light, Dark, Blue, Green, Pink, Red)
• Custom Theme Creator - Create custom color themes and save them locally
• Settings Menu - Configure game settings and preferences
• Leaderboard - View top players and their completion times
• XP/Level System - Earn XP points and track player progression
• Live Keystroke Indicator - Display recent directional inputs during gameplay
• Stopwatch Display - Real-time countdown during maze solving

GAME MECHANICS
--------------
• Difficulty Levels - Easy, Medium, Hard gameplay options
• Graphics Quality Settings - Low, Mid, High visual quality presets
• Player Position Tracking - Track player location within the maze
• End Goal Marker - Visual indicator for maze exit location
• Ghost Enemies - Moving obstacles in the maze (unlockable via shop)
• Move History - Record all player movements for replay analysis
• Breadcrumb Trails - Subtly darkened tiles show previously visited cells
• Minimap (Large Mazes) - Canvas-based overhead view for mazes larger than screen
• Challenge Links (URL Parameters) - Share exact maze seeds via URLs (auto-loads with no menu clicks)
  - Format: ?seed=<seed>&size=<difficulty>
  - Auto-authenticates if username saved in localStorage
  - Bypasses menu and loads maze directly
• Maze Generation Algorithms - Multiple generation styles with different characteristics
  - Recursive Backtracker: Long winding corridors, few dead ends, gets you lost
  - Prim's Algorithm: Short paths, many dead ends, organic branching feel
  - Spiral/Pattern: Predictable visual patterns, visual puzzle element
• Pause Functionality - Pause maze (blurs display, stops timer) to prevent cheating
  - Keyboard: Press ESC or Spacebar to pause/resume
  - UI: Pause button visible during gameplay
• Export Replay - Download completed runs as JSON files
  - Share replays with friends via file transfer
  - Import Replay feature to watch friends' solutions

MOBILE/TOUCH INPUT
------------------
• Touch/Swipe Controls - Detect swipe gestures for maze navigation (FULLY IMPLEMENTED)
  - Minimum 30px swipe distance (prevents accidental taps)
  - Swipe Up: Move player up
  - Swipe Down: Move player down
  - Swipe Left: Move player left
  - Swipe Right: Move player right
  - Works seamlessly with continuous movement system
• On-Screen D-Pad - Optional overlay with arrow buttons for tap controls
  - 4-button directional pad (↑ ↓ ← →)
  - Toggle option in settings
  - Large touch targets optimized for mobile (50px+ buttons)
• Mobile Optimization - Full-screen gameplay on phones/tablets
  - Auto-detect portrait orientation
  - Responsive grid sizing based on viewport
  - No keyboard requirement for complete gameplay
  - Install as app on home screen with PWA support

ACCESSIBILITY & UX
------------------
• Responsive Design - Mobile-friendly interface (optimized for 480px+)
• Logout Functionality - Sign out and return to login screen
• Documentation Link - Access online documentation via docs button
• Back Button - Exit maze and return to main menu
• Input Sanitization - Security validation for usernames and URLs
• HTML Entity Encoding - XSS prevention for user-entered data
• Keyboard Shortcuts - Hotkeys for common actions
  - ESC: Pause/Resume game
  - SPACE: Pause/Resume game
  - R: Restart current maze
  - Arrow Keys / WASD: Movement controls

PROGRESSION & ECONOMY
---------------------
• XP/Leveling System - Earn XP from completing mazes
• Shop (Cosmetics & Themes) - Unlock skins, avatars, and exclusive themes with XP
• Badge/Achievement System - Unlock badges for specific milestones
  • Speed Demon: Solve hard maze in under 30 seconds (+100 XP)
  • Pacifist: Win without power-ups
  • Marathon: Accumulate 10,000 total steps (+50 XP)
• Step Counter - Track lifetime steps across all games

PERSISTENCE & DATA
-------------------
• LocalStorage Integration - Save username, settings, XP, and theme preferences
• Replay Management - Store and retrieve saved game sessions
• Leaderboard Storage - Persist high scores and rankings
• Game Settings Caching - Remember user's difficulty and graphics preferences
• Custom Theme Persistence - Save custom theme configurations
• Achievement Tracking - Store unlocked badges and achievements

SECURITY
--------
• Session Token Generation - Persistent session tracking with random IDs
• Input Sanitization - Protection against XSS attacks
• JSON Validation - Safe parsing of imported maze files
• URL Validation - Prevent javascript: and data: protocol attacks
• Color Validation - Restrict to valid hex color values
• Username Filtering - Max 32 characters, HTML special character removal

AUDIO & SOUND (FULLY IMPLEMENTED)
---------------------------------
• Sound FX Manager - Synthesized 8-bit style audio (NO external audio files!)
  - Uses Web Audio API for real-time synthesis
  - Generates all sounds programmatically
  - Perfect for offline PWA deployment
• Step Sound - 400Hz sine wave descending to 100Hz (soft blip)
• Wall Bump - 100Hz square wave descending to 50Hz (dull thud)
• Victory Jingle - C Major arpeggio (C4, E4, G4, C5) triangle wave
  - Plays rapidly with fade-out envelope
  - 100ms spacing between notes
• Audio Context - Automatically resumes on user interaction
• Mute Toggle - Built-in toggleSound() function
  - Persistent audio settings via soundEnabled variable
• Synthesized Audio Advantages:
  - No file dependencies - everything in HTML
  - Tiny code footprint (~1KB)
  - Works perfectly offline
  - No licensing issues

VISUAL & EFFECTS
----------------
• Particle Effects - Confetti burst on victory, spark animations on level-up
• CSS Animations - Smooth transitions for UI and gameplay elements
• Color Themes - 6 presets + unlimited custom color creation
• Pause Visual Feedback - Blur effect on paused maze, visible pause overlay
• Loading Animations - Spinner/indicator during maze generation
• Bootstrap Icons - Professional 1000+ icon library (v1.13.1) for all UI elements
  - Replaces emojis for professional, scalable, and accessible iconography
  - Consistent visual language throughout the interface

PWA (PROGRESSIVE WEB APP)

• Offline Play - Service Worker enables 100% offline functionality
• Install to Home Screen - Add game icon directly to mobile home screen
• App Manifest - Complete PWA metadata and configuration
• Network-First Caching - Intelligent cache strategy for fast loading
• Icon Shortcuts - Quick access to Daily Challenge and Random Maze

-------------------

• Grid Rendering System - CSS Grid-based maze visualization
